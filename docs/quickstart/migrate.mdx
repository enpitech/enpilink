Already have an app built with the [OpenAI Apps SDK](https://developers.openai.com/apps-sdk/) or [`@modelcontextprotocol/ext-apps`](https://github.com/modelcontextprotocol/ext-apps)? The enpilink [Skill](/devtools/skills) can migrate it for you — it teaches your coding agent the full enpilink framework so it can scaffold, rewrite, and wire up your project automatically.

## Install the skill

<CodeGroup>
```bash npm
npx skills add enpitech/enpilink -s enpilink
```
```bash pnpm
pnpm dlx skills add enpitech/enpilink -s enpilink
```
```bash yarn
yarn dlx skills add enpitech/enpilink -s enpilink
```
```bash bun
bunx skills add enpitech/enpilink -s enpilink
```
```bash deno
deno run -A npm:skills add enpitech/enpilink -s enpilink
```
</CodeGroup>

## Prompt your agent

```
Migrate my app to /enpilink
```

That's it! Once migrated, keep using the [🧠 Skill](/devtools/skills) for ongoing development — brainstorming ideas, designing views and tools, running dev server, debugging, deploying, and more.
