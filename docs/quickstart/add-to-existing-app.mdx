enpilink contains 2 main packages that can be used together or independently.

## The packages

- **`enpilink/server`**: A drop-in replacement for the official MCP SDK that adds view registration and type inference capabilities.
- **`enpilink/web`**: A React library providing hooks, components, and the runtime glue to render your views inside the host's iframe environment.

## Option 1: use `enpilink/server` (Recommended)

<Card title="Use enpilink/server" icon="server" href="/quickstart/add-to-existing-app/server">
  **Best if**: you're using the official TypeScript MCP SDK and want to add rich UI views with full type safety.
</Card>

## Option 2: use `enpilink/web` only

<Card title="Use enpilink/web only" icon="browser" href="/quickstart/add-to-existing-app/web">
  **Best if**: you already have an MCP server in a non-TypeScript runtime and only want React hooks for UI components.
</Card>
