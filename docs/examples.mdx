enpilink ships two reference apps in the [`examples/`](https://github.com/enpitech/enpilink/tree/main/examples)
directory of the repo. Both are open source, account-free, and run entirely on
your machine — clone the repo, `pnpm install`, and `pnpm dev` in either folder.

## Kitchen Sink

The **kitchen-sink** (a fictional store called *Northwind*) is the showcase app.
It exercises **every enpilink framework feature and all five mcp-ui interaction
types** — `tool`, `prompt`, `link`, `intent`, and `notify` — across 9 tools and 9
views, with deterministic mock data so demos reproduce exactly.

It's also the basis for the `demo` scaffolder template, so `npm create enpilink`
gives you this app as a starting point.

```bash
cd examples/kitchen-sink
pnpm install
pnpm dev          # Console + HMR
pnpm dev:tunnel   # account-free public srv.us tunnel for Claude / ChatGPT
```

## Connecting an example to Claude or ChatGPT

1. Run `pnpm dev:tunnel` (kitchen-sink) — enpilink opens an account-free
   [srv.us](/quickstart/test-your-app) tunnel and prints a public `/mcp` URL.
2. In **Claude** → Settings → Connectors → **Add custom connector**, paste the
   printed `https://<hash>.srv.us/mcp` URL. (In **ChatGPT**, use Settings →
   Connectors → Create.)
3. For the kitchen-sink, paste the contents of
   [`examples/kitchen-sink/specs/SYSTEM_PROMPT.md`](https://github.com/enpitech/enpilink/blob/main/examples/kitchen-sink/specs/SYSTEM_PROMPT.md)
   into your Claude project instructions so the assistant knows which tools to call.
